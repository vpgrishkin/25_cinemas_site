Что в кино Описание API

Взаимодействие с API происходить в формате JSON.
1. Чтобы получить список фильмов, необходимо отравить GET запрос на адрес: .../movies
Описание ответа
movie_title — название фильма;
details. cinemas_count — количество кинотеатров;
details. movie_url — ссылка на описание на сайте afisha. ru.

2. Чтобы получить описание фильма, необходимо отравить POST запрос на адрес: .../movie_info
Описание запроса
{
"movie_url": movie_url,
"movie_title": movie_title
}
movie_url — ссылка на описание фильма на afisha. ru;
movie_title — название фильма.

Пример запроса
{
"movie_url": "https://www. afisha. ru/movie/229807/",
"movie_title": "Большой"
}

Пример ответа
{
"description": "Амбициозная драма про балет, станки и пуанты",
"image": "https://s5.afisha. ru/MediaStorage/97/54/731f81fc013443cca7a5ff2a5497.jpg",
"rating": 7.923
}

Настоящий сайт создан в исключительно учебных целях. Все сведения о фильмах взяты с сайтов Афиша и Кинопоиск.
